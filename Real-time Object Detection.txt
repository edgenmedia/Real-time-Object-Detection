Real-time Object Detection using YOLOv8

An AI-powered Computer Vision system for detecting and classifying objects in real time using YOLOv8, optimized for speed, accuracy, and real-world deployment.

📌 Project Overview

Real-time Object Detection using YOLOv8 is a deep learning–based Computer Vision application that identifies and localizes multiple objects in images and live video streams.

Built on YOLOv8 (You Only Look Once), the system performs single-stage detection, making it fast enough for real-time use cases such as surveillance, autonomous systems, and smart analytics.

This project demonstrates production-ready ML engineering and practical Computer Vision expertise.

🎯 Project Objectives

To implement real-time object detection using YOLOv8

To detect and classify multiple objects simultaneously

To achieve high inference speed with strong accuracy

To demonstrate industry-grade ML model deployment

To apply Computer Vision in real-world scenarios

🚀 Features

🎥 Real-time detection from webcam or video

🖼️ Image and video file inference

⚡ High-speed inference using YOLOv8

🧠 Pre-trained & custom-trained model support

🔧 Easy deployment & scalability

🧠 What is YOLOv8?

YOLOv8 (You Only Look Once) is a state-of-the-art real-time object detection model that performs detection in a single pass through the neural network.

Key benefits:

Single-stage detection

High FPS (Frames Per Second)

Excellent accuracy–speed tradeoff

Production-ready architecture

📋 Requirements

Python 3.8+

YOLOv8 model weights

GPU (optional but recommended)

Install dependencies:

pip install -r requirements.txt

▶️ How to Run
Webcam Detection
python main.py --source webcam

Image Detection
python main.py --source image --path data/image.jpg

Video Detection
python main.py --source video --path data/video.mp4
